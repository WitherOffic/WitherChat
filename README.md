# WitherChat

WitherChat — настольное приложение для Windows, предназначенное для чтения и модерации чата трансляции в реальном времени.

Проект изначально создавался для личного использования и развивается как удобный самостоятельный чат для стримов.

Текущая версия: `0.3.0`.

## Возможности

- подключение аккаунта через OAuth;
- чтение и отправка сообщений в реальном времени;
- переключение между каналами;
- поиск по сообщениям и локальные журналы чата;
- инструменты модерации;
- отображение значков и сторонних эмоутов;
- настраиваемый браузерный оверлей;
- работа в области уведомлений после закрытия окна;
- режим отображения поверх остальных окон;
- светлая и тёмная темы;
- русский и английский интерфейс.

## Требования для запуска

- Windows 10 или новее;
- для готовой self-contained сборки отдельная установка .NET не требуется;
- для обычной framework-dependent сборки требуется .NET Desktop Runtime 8.

## Сборка

Для сборки требуется .NET 8 SDK.

```powershell
dotnet restore WitherChat.sln
dotnet build WitherChat.sln --no-restore
```

Release-сборка:

```powershell
dotnet build WitherChat.sln -c Release --no-restore
```

## Локальные данные

Настройки, токены, журналы и диагностические файлы хранятся только в локальном профиле пользователя. Секретные токены не выводятся в интерфейс и не записываются в журнал приложения.

Название приложения, продукта и издателя: **WitherChat**.
